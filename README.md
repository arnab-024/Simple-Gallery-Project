# 📸 Picsum Gallery

A responsive image gallery app built with React.js that fetches random photos from the Picsum Photos API with pagination and dark/light mode support.

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- Axios
- Vite

## ✨ Features
- Fetches 30 images per page from Picsum Photos API
- Previous / Next pagination (pages 1-10)
- Shows author name for each photo
- Dark / Light mode toggle
- Loading state while images are being fetched
- Click on the image to visit the original photo

## 🚀 Run Locally
git clone <your-repo-url>

cd Simple-Gallery-Project

npm install

npm run dev
